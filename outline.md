# 涂料重量计算器 - 项目概述

## 项目简介
这是一个专业的工业涂料重量计算工具，为制造业和涂装行业提供精确的涂料用量计算服务。基于用户提供的参数（面积、厚度、密度），自动计算底漆、面漆及固化剂的精确重量。

## 核心功能模块

### 1. 主计算器页面 (index.html)
- **Hero区域**：简洁的标题和工具介绍
- **计算器界面**：
  - 左侧：参数输入区域
  - 右侧：计算结果展示
- **实时计算逻辑**：基于输入参数自动计算
- **结果可视化**：使用图表展示涂料配比

### 2. 示例问题页面 (examples.html)
- **预设示例**：包含用户提供的标准计算案例
- **计算步骤展示**：详细的计算过程分解
- **结果对比**：多个示例的对比分析

### 3. 使用说明页面 (guide.html)
- **功能介绍**：详细说明各项功能
- **计算公式**：展示背后的数学原理
- **常见问题**：FAQ解答

### 4. 历史记录功能
- **本地存储**：保存最近计算记录
- **记录管理**：查看、删除、重新加载历史数据

## 技术架构

### 前端技术栈
- **HTML5**：语义化结构
- **CSS3 + Tailwind**：响应式样式
- **JavaScript ES6+**：核心计算逻辑
- **Anime.js**：动画效果
- **ECharts.js**：数据可视化
- **Shader-park**：背景视觉效果

### 核心算法
```javascript
// 涂料重量计算公式
paintWeight = (area * thickness * density) / 1000
// 固化剂重量计算
curingAgentWeight = paintWeight * (ratio / (ratio + 1))
```

## 用户体验设计

### 交互流程
1. 用户输入参数（面积、厚度、密度）
2. 实时验证输入数据的有效性
3. 点击计算或自动计算结果
4. 动画展示计算结果
5. 保存到历史记录

### 视觉设计
- **工业美学**：体现专业性和精确性
- **数据可视化**：直观的图表展示
- **响应式布局**：适配各种设备
- **动画反馈**：增强用户交互体验

## 数据精确性
- **数值精度**：保留4位小数
- **单位统一**：公斤(kg)、平方毫米(mm²)、毫米(mm)、千克/升(kg/L)
- **验证机制**：输入范围检查和错误提示

## 部署和维护
- **静态网站**：易于部署和维护
- **本地存储**：无需服务器端支持
- **离线可用**：支持离线计算功能
